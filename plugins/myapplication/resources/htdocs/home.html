Welcome to your new Web Application

This application is built using a web application framework called Aurora. This provides a build/compilation system with modular plugins. Default plugins are provided for various things like HTTP handling or Client side widgets. Widgets are used to put dynamic data on html pages and data access is handled via a simple publish/subscribe system and websockets. You can find out more over at /docs#aurora

Closure compiler is used for code quality as well as production obfuscation.

This application gets its state management engine and many functional widgets from the Recoil project. Recoil is a Funcational Reactive Programming (FRP) engine for Javascript. It comes with a library of widgets that are suitable for use with functional programming. FRP is useful as it solves issues with the observer pattern and allows the programmer to express logic without considering how data was obtained, or considering time.

You can change this page by editing plugins/myapplication/home.html. When you read the docs, you can learn about plugins, pages and widgets.
